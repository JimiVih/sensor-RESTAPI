Guide to get you started

!GOOD TO KNOW BEFORE INSTALLATION!
-Python version used was 3.13.x

Installation
1) Download the .zip file and extract the content to the folder of your choice
2) Open the folder in Konsole/cmd, or in vsCode and open up the terminal in VScode
3) Create a virtual environment `python -m venv <folder name (ex. .venv)>`
    3.1) if you have, let's say Python version 3.14.x installed and want to use older version inside the venv (in this case version 3.13.x);
         you can write `python3.13 -m venv <folder name>`
4) Change your current python to the python of the venv (or activate the virtual environment) `source ~/<folder_name>/bin/activate`
